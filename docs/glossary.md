# Glossary

- **Photo Stack**: The main interactive interface component where images are presented as a stack for sorting.
- **Photo Card**: A single image element within the photo stack.
- **Tinder Mode**: Standard 2-way sorting (Left/Right).
- **Tinder Plus Mode**: Extended 4-way sorting (Left/Right/Up/Down).
- **Dry Run Mode**: A safety state where actions are simulated and logged but no actual filesystem changes are made.
- **Intent**: A user's decision to move/copy/trash a photo, recorded before execution.
- **Source Root**: The directory where unsorted photos are scanned from.
- **Destination Root**: The base directory where sorted photos are moved to (Keep, Review, etc.).
- **Metadata Drawer**: The overlay that reveals detailed photo information and EXIF data.
